Tag aside

a tag ASIDE é utilizada para marcar conteúdo complementar ao principal do site.

Exemplos: lista de posts mais visitados, informações extras como a descrição de um termo usado no artigo e mais (quem viu isso viu tambem...) (produtos mais procurados)

article

A tag article é uma seção (parte) do site que não precisa do site todo para fazer sentido, ou seja, é uma região auto-explicativa.

O aria-labelledby é um atributo utilizado quando se quer evitar repetições de palavras próximas no leitor de tela, ex: se você colocar um título 'celular' e uma descrição aria-label = 'celular', o leitor de tela lerá duas vezes a palavra celular causando um desconforto. Então, utilizamos um ID no título e um aria-labelledby com, assim o leitor de tela lerá somente uma vez.

Main

tag Ol, UL, DL e LI

a tag UL é utilizada para criar listas não ordenadas (unodered list). ex:

  • Menus de um site
  • lista de topicos mais visitados
  • lista de ingredientes

OL é uma lista ordenada (ordered list). ex:
